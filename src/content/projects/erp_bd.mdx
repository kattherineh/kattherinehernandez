En este proyecto, trabajé con SQL Server para implementar un sistema de gestión de bases de datos que 
incluye el uso de DDL (Data Definition Language), DML (Data Manipulation Language), procedimientos almacenados 
y triggers. A través de esta experiencia, desarrollé las siguientes habilidades:

- Diseño de Esquemas de Base de Datos
>Aprendí a crear y modificar estructuras de bases de datos utilizando DDL, lo que incluye la creación de tablas, 
>índices y relaciones entre ellas.

- Manipulación de Datos
>Mejoré mis habilidades en DML, permitiéndome realizar operaciones de inserción, 
>actualización y eliminación de datos de manera eficiente, garantizando la integridad y 
>consistencia de la información. 

- Desarrollo de Procedimientos Almacenados 
>Adquirí experiencia en la creación de procedimientos almacenados, lo que me permitió encapsular lógica 
>de negocio compleja y mejorar el rendimiento de las consultas al reducir la necesidad de enviar múltiples 
>solicitudes a la base de datos. 

- Implementación de Triggers
>Aprendí a implementar triggers para automatizar acciones en respuesta a eventos específicos en la base de datos, 
>lo que contribuyó a mantener la integridad de los datos y a aplicar reglas de negocio de forma efectiva. 

- Optimización de Consultas 
>Desarrollé habilidades en la optimización de consultas SQL, aprendiendo a identificar cuellos de botella 
>en el rendimiento y a aplicar mejores prácticas para mejorar la eficiencia de las operaciones. 

- Manejo de Transacciones 
>Comprendí la importancia de las transacciones en SQL Server, lo que me permitió implementar mecanismos 
>de control para garantizar que las operaciones de la base de datos sean atómicas y consistentes. 

Este proyecto no solo fortaleció mis habilidades técnicas en SQL Server, sino que también me brindó 
una comprensión más profunda de cómo las bases de datos pueden ser utilizadas para resolver 
problemas empresariales complejos.
