Este proyecto contiene endpoints para la lógica de negocios de una aplicación de contratación de 
servicios de transporte similar a Uber. 

Fue mi primer acercamiento al framework Spring Boot, lo que me permitió expandir mis conocimientos 
sobre la arquitectura cliente-servidor. Además, desarrollé un mayor interés en lenguajes de programación 
tan robustos como Java. 

Participar en este proyecto me ayudó a adquirir competencias en el desarrollo backend 
con Spring Boot y a mejorar habilidades blandas como la gestión de emociones al trabajar bajo presión 
y la inteligencia emocional. 

Repositorio: https://github.com/kattherineh/Backend-Proyecto-Lenguajes-IS513-UBER.git 

## Tecnologías utilizadas:  

- Framework Springboot – Java. 
- Visual Studio Code
