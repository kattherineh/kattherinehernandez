### Objetivo: Consumo de servicios REST API de Springboot desde Laravel. 

Una de las partes más cruciales en un proyecto de desarrollo web que sigue el modelo cliente-servidor 
es el consumo de los servicios backend desde el frontend. 

En esta ocasión, tuve la oportunidad de compartir con mis compañeros sobre este tema y 
desarrollamos un ejercicio práctico. Esta actividad me permitió profundizar en el funcionamiento 
del modelo cliente-servidor, utilizando Guzzle como cliente para consumir servicios de manera efectiva. 

Repositorio: https://github.com/kattherineh/Consumo-de-Web-Services-desde-Laravel-PHP.git

## Tecnologías utilizadas:  

- Framework Laravel – PHP 
- Visual Studio Code 
- Framework Springboot – Java 
- Libreria Guzzle
