Este proyecto consistió en la implementación de un sistema básico para Autenticación de usuarios
utilizando Firebase como gestor de los mismos.

Para iniciar sesión, el usuario debe estar registrado. El usuario solo podrá realizar acciones en el sistema si se encuentra activo,
para ello pasará por un proceso de activación.

A través de esta experiencia, adquirí varias habilidades clave: 

- Despliegue de aplicaciones web utilizando los servicios de Azure.
>Entre esos servicios destacan: App services plan, App services, Azure SQL, Azure function apps,
>storage accounts (utilizando los queue storage), entre otros.

- Desarrollo de servicios con Fast API 
>El backend fue desarrollado con este framework de Python.

- Interfaz gráfica con Next.js
>EL frontend fue desarrollado con ese framework de Node.js.
