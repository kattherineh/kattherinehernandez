Al finalizar el curso de Git, he desarrollado una comprensión sólida de esta herramienta esencial en el 
desarrollo de software. Aprendí a instalar y utilizar Git de manera efectiva, tanto en entornos gráficos 
como en la terminal, lo que me permite gestionar proyectos de manera más organizada. 

Entender conceptos clave como ramas, commits y flujos de trabajo me ha permitido integrar Git en mi 
práctica diaria. Además, mi experiencia con GitHub me capacita para colaborar en proyectos de código abierto 
y presentar mis trabajos de forma profesional. 

Este curso no solo me ha enseñado a manejar Git y GitHub, sino que también me ha proporcionado habilidades 
para resolver problemas comunes que pueden surgir al usar estas herramientas. Con esta base, estoy preparada 
para contribuir de manera efectiva en equipos de desarrollo y adaptarme a las prácticas comunes en la industria. 

Estoy emocionada por aplicar mis conocimientos en entornos colaborativos, optimizando procesos y mejorando 
la calidad del código en los proyectos en los que participe.
